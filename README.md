## how to run the application locally.

### install process command

- npm init -y
- yarn add express
- yarn add -D typescript
- tsc-init
- yarn add -D @types/node
- yarn add -D @types/express
- yarn add -D nodemon
- tsc -w

#### package.json

- start:dev: 'nodemon ./dist/server.ts'
